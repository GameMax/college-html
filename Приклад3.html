HTML - мова розтки гіпертексту

HTML (HyperText Markup Language) - мова розмітки гіпертексту. HTML відповідає за те, що користувач бачить в певній послідовності на веб-сторінці текст, графіку, анімацію, відео. Найважливішою рисою Web-сторінок є гіпертекстові посилання.
