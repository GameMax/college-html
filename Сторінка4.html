Сайтобудування

Рекомендації:

1) Вивчіть мову html та технологію css.

2) Використовуйте візуальні редакторів HTMLPad, Adobe Dreamweaver та інші для автоматизації процесу написання html і css коду.

3) Для створення динамічного сайту можна використати движки* наприклад:

* Система управління сайтом (CMS - Content Management System, движок для сайту) - програмне забезпечення, що дозволяє керувати вмістом і структурою сайту.

Розміщення сайту

Після того, як сайт створений, його потрібно розмістити в глобальній мережі Інтернет.

Для розміщення сайту необхідно вибрати домен і придбати його. Доменне ім'я – це майбутнє ім'я вашого сайту в мережі Інтернет. Далі для розміщення сайту необхідно вибрати хостинг. Хостинг – дисковий простір, орендований на серверах в Інтернеті, на якому буде розташовуватися вміст вашого сайту.

Приклади сайтів для оренди хостингу і підбору доменних імен: X-HOST, DNS.COM.UA, Хостинг Украина, UH.UA.
